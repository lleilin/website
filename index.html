Multithreaded Filesystem
I developed a multithreaded filesystem in C++ using Boost libraries, allowing concurrent read/write operations and boosting performance through parallel task execution. The design incorporated Websockets for secure and scalable communication between filesystem components and implemented robust error handling for enhanced stability.
C++
MapReduce Framework Implementation
In EECS 485, I built a distributed MapReduce framework in Python, inspired by Google's design, enabling parallel processing of large datasets across computer clusters. The framework efficiently distributes and executes map and reduce tasks, handling complex computations with fault tolerance for increased reliability.
Python
